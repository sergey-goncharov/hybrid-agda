Generalised Duration Monad in Cubical Agda
------------------------------------------

* The code can be combiled with the  Agda proof assistant, version 2.6.1-96d0dd0 
This can be installed by running 

$ git clone https://github.com/agda/agda.git
$ git checkout 96d0dd00b696b381d87972ec78c4de31ff20b414

and using the instructions from https://agda.readthedocs.io/en/latest/getting-started/installation.html

* The cubical library is required 

$ git clone https://github.com/agda/cubical.git
$ git checkout 43d5fcfbe1a3fe830141745442548b1b4ee25f81

* The most recent version of our library can be found under https://git8.cs.fau.de/
software/duration-monad-agda/tree/master
